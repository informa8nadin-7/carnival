# Эхо-бот на aiogram

Простой Telegram-эхобот, который повторяет пользователю его сообщения.

## Установка и запуск

1. **Клонируйте проект** (или скопируйте папку с кодом).
Обновить `src/bot/config.py` и `/.env.example`: добавить `POLZA_API_KEY`, `POLZA_MODEL`, `POLZA_BASE_URL`, `POLZA_HISTORY_MESSAGES_LIMIT`.
Добавить `src/bot/services/polza_chat.py`: payload, ограничение истории, HTTP POST (standard library), парсинг `choices[0].message.content`.
Добавить `src/bot/routers/chatgpt.py`: режим FSM, обработчики `/chatgpt`, диалоговые сообщения, `/chatgpt_off`.
Подключить `chatgpt_router` в `src/bot/routers/__init__.py`.
Обновить `src/bot/routers/echo.py`: не отвечать, когда активен `ChatGPTStates.active`.
Обновить `src/bot/services/text.py` (help) и при согласии пользователя — клавиатуру в `src/bot/keyboards/common.py`.
Добавить тесты для `src/bot/services/polza_chat.py` и запустить `pytest`.


2. **Создайте и активируйте виртуальное окружение** (из корня проекта):

```bash
python -m venv .venv
.venv\Scripts\activate
```

3. **Установите зависимости**:

```bash
pip install -r requirements.txt
```

4. **Создайте файл `.env` на основе `.env.example`**:

```bash
copy .env.example .env
```

Впишите в `BOT_TOKEN` токен вашего бота из BotFather.

**Альтернативно** вы можете использовать мастер настройки, который поможет создать `.env` и ввести токен:

```bash
python setup.py
```

5. **Запустите бота** (из корня проекта):

```bash
python -m src.bot
```

## Мастер настройки

В проекте есть интерактивный мастер настройки `setup.py`, который:

- Проверяет наличие файла `.env` и создаёт его при необходимости.
- Запрашивает токен бота у пользователя.
- Сохраняет токен в `.env`.
- Предлагает сразу запустить бота.

Запустите мастер командой:

```bash
python setup.py
```

## Команда /chatgpt

Бот поддерживает команду `/chatgpt`, которая переводит пользователя в режим диалога с ИИ через Polza.ai (OpenAI‑compatible API). Особенности:

- Сохраняет контекст из последних 5 сообщений.
- Для работы требуется API‑ключ Polza.ai (указать в `.env` как `POLZA_API_KEY`).
- Выход из режима — команда `/chatgpt_off`.

Пример использования:
1. Отправьте боту `/chatgpt`.
2. Отправляйте любые текстовые сообщения — бот будет отвечать с помощью ИИ.
3. Завершите диалог командой `/chatgpt_off`.

## Команда /image

Бот поддерживает команду `/image <описание>`, которая генерирует изображение по текстовому описанию с помощью AI (Polza.ai). Особенности:

- Использует модель `gpt-image-1` (по умолчанию).
- Возвращает сгенерированное изображение как фото.
- Для работы требуется API‑ключ Polza.ai (указать в `.env` как `POLZA_API_KEY`).

Пример использования:
1. Отправьте боту `/image кот в шляпе`.
2. Бот отправит сгенерированное изображение кота в шляпе.
